Создать таблицы бренда и машин:

CREATE TABLE brands (
  id INTEGER primary key autoincrement,
  name VARCHAR(50)
);

CREATE TABLE cars (
  id INTEGER primary key autoincrement,
  model VARCHAR(50),
  release_year INTEGER,
  car_brand INTEGER,
  FOREIGN KEY (car_brand) REFERENCES brands(name)
)


Создание брендов и машин (заполнение таблиц):

Обновление бренда и машины по id:
